CiscoNote1

上了一节 Cisco之后发现不做笔记真的不行啊….

前提

LAN ：局域网，一切的智能化终端设备都处于局域网。 手机 电脑 ipad 智能手表（具备网卡） – 家庭网络、企业网、园区网、校园网
MAN：城域网，
WAN：广域网，跨越地理位置环境的综合性网络，目前世界上最大的广域网是因特网–Internet

上网所需要满足的条件—
①物理条件：传递信号的介质
②协议条件：TCP/IP协议栈 IP协议– IP地址（ 在网络中用于唯一标识一台网络设备 。）

IP地址的分类—首位 中间 中间 末尾
A类：首位的范围在1-126之间，10.1.1.1
B类：首位的范围在128-191之间，172.16.1.1
C类：首位的范围在192-223之间，192.168.131.206

针对每一个IP地址，又细分为—
①网络位：网络的标识，范围。
②主机位：主机的标识，范围中的具体一个点。

A类：10.100.100.100 网络位：10.0.0.0 网络中的地址数量：256^3=2^24
B类：172.16.100.100 网络位：172.16.0.0 网络中的地址数量：256^2=2^16
C类：192.168.10.100 网络位：192.168.10.0 网络中的地址数量：256=2^8

子网掩码：配置IP地址，后面一定要跟上子网掩码，否则无效；成对出现的，用于标识IP地址所处的网络范围 。
默认的子网掩码：
A类：255.0.0.0
B类：255.255.0.0
C类：255.255.255.0

网络号：拿IP地址和子网掩码做一个与运算，得到网络号，代表一个网络范围

【三大基本网络设备】
Router：路由器，连接广域网的设备 。
Switch：交换机，连接局域网的设备 。
Modem：调试解调器 ，光猫，将光信号和数字信号进行相互转换

进入Cisco仿真操作

【网络设备的三大基本模式】
Router> –用户模式，衡量一台设备是否正常运行
Router>enable
Router# –特权模式，只能查看代码，不能配置代码，用于通信测 试，ping指令
Router#configure terminal
Router(config)# –配置模式，可以进行代码的设置
Router(config)#hostname ND-NB666 —更改设备名称
exit –返回上一级
end –不管处于什么模式，马上回到特权模式

【脚本语言的特点】
①一行代码，一个结果
②代码可以缩写
③代码不区分大小写
④可以使用？提示代码
⑤可以使用TAB键自动补全代码
⑥错误的地方可以准确提示

ND-NB666(config)#int f0/0 -进入物理接口，编号f0/0
ND-NB666(config-if)#no shutdown –打开接口
ND-NB666(config-if)#ip address 10.1.1.1 255.0.0.0 –配置IP地址，后面跟上掩码
Router(config-if)#no ip address –删除代码是相同模式下，相应命令前加no(no ip address不用加ip地址和掩码)

关于Markdown语法的学习笔记

一、标题

使用#符号创建标题，标题的级别由#的个数决定。示例：
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
效果：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

二、文本格式

  • 加粗：使用两个星号（**）或两个下划线（__）包围文本。示例：
    **粗体文本**或__粗体文本__
    效果：
    粗体文本 粗体文本
  • 倾斜：使用一个星号(*)或一个下划线（）包围文本。示例：
    *斜体文本*或_斜体文本_
    斜体文本 斜体文本
  • 代码块：使用三个反单引号（```）包围文本。示例：

三、列表

无序列表

使用星（*）号、加号（+）或减号（-）包围列表项。示例：
* 第一项
+ 第二项
- 第三项
效果：

  • 第一项
  • 第二项
  • 第三项
有序列表

使用数字和点号(.)创建有序列表。示例:
1. 第一项
2. 第二项
效果:

  1. 第一项
  2. 第二项

四、链接

  • 使用方括号（[]）包围链接文本，并在后面添加一个冒号(:)和链接地址。示例:
    [必应]（https://cn.bing.com/）
    效果：
    必应

五、图片

  • 使用感叹号(!)和图片地址创建图片链接。示例：
    ![蜡笔小新](https://pic1.zhimg.com/80/v2-af290cd4c2e5b5c9e3460559f86db2ca_720w.webp?source=1940ef5c)
    效果:

六、引号

使用大于号(>)包围引用文本。示例:
> 引用文本

引用文本

七、表格

使用表格语法创建表格。示例:
| 表头 | 表头 |
| ---- | ---- |
| 单元格 | 单元格 |
效果:

表头 表头
单元格 单元格

八、删除线

使用波浪号(~)包围文本。示例：
~~删除线~~
效果:
删除线

九、下划线

示例:
<u>下划线</u>
效果:
下划线
